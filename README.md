## الگوریتم‌های پیاده‌سازی‌شده

### 1. Brute Force (جستجوی کامل)
- همه جایگشت‌های ممکن را بررسی می‌کند
- پیچیدگی: **O(n!)**
- تضمین: جواب **بهینه دقیق**
- محدودیت: فقط برای `n ≤ 12`

### 2. Greedy Search (حریصانه)
- در هر گام نزدیک‌ترین شهر ندیده را انتخاب می‌کند
- پیچیدگی: **O(n²)**
- سریع اما ممکن است کیفیت پایینی داشته باشد

### 3. Hill Climbing (صعود تپه)
- با عملگر **2-opt** مسیر را بهبود می‌دهد
- از **Random Restart** برای فرار از بهینه محلی استفاده می‌کند
- پیچیدگی: **O(restarts × n²)**

### 4. Simulated Annealing (بازپخت)
- با کاهش دما، حرکت‌های بد را با احتمال قبول می‌کند
- پیچیدگی: **O(iterations × n)**
- معمولاً بهترین نتیجه را در زمان معقول می‌دهد

### 5. Genetic Algorithm (ژنتیک)
- از **Order Crossover (OX)** و جهش جابجایی استفاده می‌کند
- پیچیدگی: **O(generations × population × n)**
- مناسب برای مسائل بزرگ

## تحلیل پیچیدگی

| الگوریتم | زمانی | فضایی | نوع جواب |
|---|---|---|---|
| Brute Force | O(n!) | O(n) | دقیق |
| Greedy | O(n²) | O(n) | تقریبی |
| Hill Climbing | O(k·n²) | O(n) | تقریبی |
| Simulated Annealing | O(iter·n) | O(n) | تقریبی |
| Genetic Algorithm | O(g·p·n) | O(p·n) | تقریبی |
